# 06三殊胜（3/3）

我们做任何一件善事时都不能离开三殊胜。本课讲述了三殊胜中的第一个：发心殊胜。发心包括不善、无记、善三种，其中善的发心又分为上、中、下三个层次。行善时要先观察自己的发心，以发大乘的菩提心为目标。

视频内容与法本内容不完全一致，为了更好地理解法义，建议学习时，视频和法本相结合。
