概述


生存的方式和生存的意义不能混为一谈，分清二者是学佛的第一个入口。本课讲述了佛陀对佛教徒生存方式的要求是少欲知足，且远离杀盗淫妄等十不善；生存的意义则应是：培养出离心，走上解脱道。
