   小编语 

《佛说稻秆经》对于学习教材第三册的《因果不虚》非常重要，所以在学习《因果不虚》正式课程前请先学习《佛说稻秆经》。

概述


本课继续讲解外因缘法里面的缘相应，通过“彼芽亦非自作，亦非他作”等，说明要区分世俗谛胜义谛（或宏观微观）去认识因果，然后通过“应以五种观彼外因缘法”等讲述外因缘法的五种原则。
